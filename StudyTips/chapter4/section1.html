​ 如上所示，登陆之后，MySQL拒绝了我查看所有数据库的命令

​ 网查解决方式如下，：

​ MySQL版本5.7.6版本以前用户可以使用如下命令：

mysql> SET PASSWORD = PASSWORD('Xiaoming250');

​ MySQL版本5.7.6版本开始的用户可以使用如下命令：

mysql> ALTER USER USER() IDENTIFIED BY 'Xiaoming250';
